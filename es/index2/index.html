FishNET: An automated relational database for zebrafish colony management

Head Developer / Internship Project / Used by Multiple Laboratories

FishNET facilitates remote tracking of individual zebrafish lines and links associated biological resources to enable robust and efficient colony management and storage of laboratory information related to zebrafish.

Filemaker was used to create a cross-platform relational database that supports multiple users and groups per organization. More than a hundred Filemaker scripts were needed in order to deliver a good user experience. Also, with the feedback of more people in my team I was able to design a convenient and fast UI. All the information provided by the user is stored in a private MySQL database.

First author resource paper:

https://doi.org/10.1371/journal.pbio.3000343
OrthoApp: Management web application for Orthodontists

Head Developer / 2019 Project / Currently in Beta

Web application created for Orthodontists. The purpose of this application is to mantain organization and standards in the dentists office. This is accomplished by keeping track of each patient their appointments and any other necessary notes. Also, keeping track of office finances and calendars.

The release of this application is schedule for December 2019 - January 2020. Some of the technologies being used for development are Vue.js, Laravel, MySQL and Amazon S3. The combination of new technologies like Vue.js and polished frameworks like Laravel gives the application a solid foundation for continous improvement.

Nutritionist app

Head Developer / 2019 Project

This application is being design by a nutritionist and Me. The purpose of this aplication is to make easier to keep track of each patient. The most important aspect of it is to make all the calculations in order to create a food menu apropiated for that patient.

The framework currently used to develop this application is Quasar, a solution based on Vue.js created to build website, desktop and mobile applications. For the back-end Firebase is being used for authentication and Firestore as database.
