HealthAdmin: Aplicación web de gestión para ortodoncistas
Desarrollador Principal
2019 Project
Actualmente en Beta

Aplicación web creada para Ortodoncistas. El propósito de esta aplicación es mantener la organización y los estándares en el consultorio del dentista. Esto se logra realizando un seguimiento de cada paciente, sus citas y cualquier otra nota necesaria. Además, realizar un seguimiento de las finanzas y los calendarios de la oficina.

Algunas de las tecnologías que se utilizan para el desarrollo son Vue.js, Laravel, MySQL y Amazon S3. La combinación de nuevas tecnologías como Vue.js y frameworks como Laravel le da a la aplicación una base sólida para la mejora continua.
